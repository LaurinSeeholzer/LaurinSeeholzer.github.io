Checkyourcover

Stushare ist eine web basierte App für Musiker, Künstler und Labels auf der ganzen Welt, welche es erlaubt Songs und Alben in einer virtuellen Spotify-App anzusehen, und neue hinzuzufügen, um sich so ein Bild zu machen wie ein bestimmtes cover auf dem Künstlerprofil aussehen würde.
Gegründet wurde Checkyourcover von Nikola Kitanovic und entwickelt von Laurin Seeholzer
Die Web-App ist free-to-use und kann unter https://checkyourcover.com genutzt werden.

Zur umsetzung des Projekts wurden die Programmiersprachen HTML, CSS und JavaScript zusammen mit dem CSS-Framework Bootstrap für das Frontend verwendet. Um die Plattform auch funktionsfähig zu machen, bediente man sich der Programmiersprache Python zusammen mit der Library Flask im Backend, welche es erlaubt Requests einfach zu empfangen, validieren/verarbeiten und mit der entsprechenden response zu antworten. Als Kern des Projekts diente die Spotify-API, um Künstler Daten von Spotify zu beziehen um so eine dynamische, virtuelle spotify-App zu programmiern.
